# ADHD Igniter

ADHDの脳科学に基づいた集中サポートアプリ / Focus igniter built on ADHD neuroscience

## コンセプト

ポモドーロ法の「25分やれ」はADHDの脳に合わない。タスクを始めること自体が一番難しい。  
ADHD Igniterは**「まず2分だけ」**という着火剤で動き出しを助ける。

## 主な機能

- 🧠 **ブレインダンプ** — 頭の中を全部書き出してから整理
- 📅 **時間軸管理** — 今日/今週/今月/いつかやるで振り分け、今日3つに絞る
- 🔥 **2分チャレンジ** — まず2分だけ強制スタート
- 🎯 **6つの集中トリガー** — 宣言・締め切り・音楽・ご褒美・1タスク集中・運動プライミング
- 📊 **適応的チェックイン** — 集中度に応じてランダムなタイミングで自己確認
- 🔥 **フレキシブルストリーク** — 1日スキップ許容の継続記録
- 🎁 **ご褒美システム** — ポイントを貯めて自分へのご褒美と交換
- 🕯️ **着火儀式** — 毎回の集中前に同じルーティンで文脈手がかりを作る

## 科学的根拠

- 実装意図（if-then）がADHDの反応抑制を改善（Gollwitzer & Brandstätter）
- 5分の有酸素運動がドーパミン受容体を活性化（Volkow et al., 2011）
- 運動が実行機能を有意改善（メタ分析 PLOS ONE 2023、PMC10434964）
- 文脈的手がかりが習慣を活性化（Wood, Annual Review of Psychology 2016）
- 変動比率報酬がエンゲージメントを向上（スキナーの強化スケジュール）

```bash
xcodegen generate
open ADHDIgniter.xcodeproj
```
